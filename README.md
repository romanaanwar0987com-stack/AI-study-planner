# AI-study-planner
🤖 AI Study Planner Agent — An intelligent study planning tool powered by LLaMA 3.3 (Groq API). Generate personalized study plans, analyze weak topics, get tips &amp; tricks, and download your plan as PDF. Built with Streamlit &amp; Python.
 
 ## 🚀 Features
- Personalized study plan generation
- Weak topics analysis
- Daily study schedule
- Tips & tricks for each subject
- Practice questions
- PDF download
- History of previous plans

## 🛠️ Tech Stack
- Python
- Streamlit
- Groq API (LLaMA 3.3)
- ReportLab (PDF)

## ⚙️ Installation

1. Clone the repository
   git clone https://github.com/YOUR_USERNAME/AI-study-planner.git

2. Install dependencies
   pip install -r requirements.txt

3. Create .env file and add your Groq API key
   GROQ_API_KEY=your_api_key_here

4. Run the app
   streamlit run app.py
